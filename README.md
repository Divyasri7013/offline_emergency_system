# offline_emergency_system
SafeNet is a web-based Offline Emergency Alert System designed to help users quickly report emergency situations, even when internet connectivity is unavailable. The platform captures incident details, geolocation data, and supporting information, storing reports locally when offline and automatically syncing them to the server once a connection is restored.

Key Features:
- Emergency incident reporting
- Real-time geolocation tracking
- Offline data storage and synchronization
- Alert management dashboard
- User-friendly and responsive interface
- Reliable operation in low-connectivity areas

Technologies Used:
- Django
- HTML5
- CSS3
- JavaScript
- MySQL
SafeNet aims to improve emergency response by providing a dependable communication platform that works seamlessly across both online and offline environments.
